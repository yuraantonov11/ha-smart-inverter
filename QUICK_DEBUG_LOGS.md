## 🚀 Швидкий Старт: Debug Потрібен Файл-розташування

### Як Увімкнути?

1. **Зайдіть у Settings (Параметри)**
2. **Клацніть на версію додатку 7 разів** (внизу)
3. **Режим Розробника активований** ✅

### Як Переглянути Логи?

1. **Settings → 🐛 Debug Logs (вже видимий)**
2. **Натисніть кнопку**
3. **Перегляньте критичні від ХЕМС**

### 📍 Де Зберігаються Файли?

```
Windows: C:\Users\[ваше_імя]\Documents\siseli_debug_logs\
Android: /storage/emulated/0/Documents/siseli_debug_logs/
```

### 🔍 Шо Логується?

- ⚠️ Кількість батареї < 35%
- 🔴 SOC критичний (< 22%)
- 📱 Конфлікти режимів (ПК ↔ Телефон)
- ⏱️ Дані застарілі > 30 хвилин
- 🌅 Вечерня захист (18:00-23:00)

### Приклад лог-запису:

```
[18:45:33] [BATTERY_RECOVERY] CRITICAL RECOVERY: SOC=34.2% < 35% → Force USB+SNU until 45%+
[18:47:15] [MODE_CONFLICT] EXTERNAL MODE CHANGE: sent=SBU, device shows=USB
```

**Для звітів про проблеми:** Скопіюйте весь вміст лог-файлу та надішліть 📧
