## Smart Inverter App v2.0.0

### Highlights
- Dashboard scrolling and repaint performance improvements.
- Menu label overflow fixes for long localized words (single-line ellipsis behavior).
- Material 3 design alignment updates across screens.
- Main shell rebuild scope reductions for smoother navigation.
- Updater version comparison improved to include build number (`+build`) to prevent missed updates.
- Automated multi-platform release workflow for Windows and Android on tag push.
